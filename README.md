# KS Eclipse

Coming Spring 2026

KS Eclipse is a modern, performance-first Shopify theme built for flexibility and speed. It includes a refined Light and Dark mode system with manual switching or automatic syncing to device preferences for a seamless experience. This free edition delivers a strong core foundation, while premium and advanced features are reserved for our premium themes.

View the demo to see the work in progress.
